# Smoke-Detection
The MQ-2 smoke sensor is sensitive to smoke and to the following flammable gases:
  -LPG
  -Butane
  -Propane
  -Methane
  -Alcohol
  -Hydrogen
The resistance of the sensor is different depending on the type of the gas.
